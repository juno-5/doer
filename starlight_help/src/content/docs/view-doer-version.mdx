## Doer server and web app version

If your organization self-hosts Doer, the server administrator will need to
[upgrade Doer][upgrade-doer] in order for you to experience the latest
improvements. You can check which [version][changelog] of the Doer server and
web app your organization is using.

Doer Cloud organizations are always updated to the latest version of Doer.

[upgrade-doer]: https://zulip.readthedocs.io/en/stable/production/upgrade.html

[changelog]: https://zulip.readthedocs.io/en/latest/overview/changelog.html

### View Doer server and web app version

<Tabs>
  <TabItem label="Doer Server 6.0+">
    <Steps>
      1. Click on the **gear** (<GearIcon />) icon in the upper
         right corner of the web or desktop app.
      1. View the version number or Doer Cloud plan in the top section of the menu.
      1. *(optional)* Click on the version number or Doer Cloud plan for additional
         details.
    </Steps>
  </TabItem>

  <TabItem label="Doer Server 4.0+">
    <FlattenedSteps>
      <NavigationSteps target="relative/gear/about-doer" />

      1. View the version number under **Doer Server**.
    </FlattenedSteps>
  </TabItem>
</Tabs>

<DoerTip>
  If the Doer server is running a custom branch, the **merge base**
  will also be shown. It indicates which official version of Doer
  the custom branch is based on.
</DoerTip>

## Doer desktop app version

The [Doer desktop app](/apps/) has an independent version from the server and
web app.

### View Doer desktop app version

<Steps>
  1. Click on the **Help** menu in the desktop app. The app version is shown in the
     dropdown: **Doer Desktop v...**
</Steps>

## Doer mobile app version

The [Doer mobile apps](/apps/) have an independent version from the server and
web app.

### View Doer mobile app version

<FlattenedSteps>
  <MobileMenu />

  1. Tap <InfoIcon /> **About Doer**.
</FlattenedSteps>
