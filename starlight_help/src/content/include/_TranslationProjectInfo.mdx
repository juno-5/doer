Doer has been translated or partially translated into dozens of
languages by Doer's amazing community of volunteer translators.
You can see which languages Doer supports [on Weblate][weblate-doer].

If you'd like to help by contributing as a translator, see the
[Doer translation guidelines][translating-doer] to get started.

[weblate-doer]: https://hosted.weblate.org/projects/doer/

[translating-doer]: https://zulip.readthedocs.io/en/stable/translating/translating.html
